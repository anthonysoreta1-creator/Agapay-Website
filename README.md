# Agapay-Website

AGAPAY is an interactive and user-friendly website designed to educate and guide users on disaster preparedness. Developed as a final project for the Introduction to Computing course, this website provides essential information, tips, and resources to help individuals and communities stay safe during natural disasters.

Features: Informative sections on different types of disasters Step-by-step preparedness guides Easy navigation with responsive design for desktop and mobile Engaging layout and visuals for better understanding

Language used:
HTML and CSS only

Objective: To raise awareness about disaster preparedness and provide practical tools and knowledge for users to effectively respond in emergency situations.
